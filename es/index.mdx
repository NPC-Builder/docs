NPC Builder es una plataforma para crear personajes no jugables (NPC) impulsados por IA que dan vida a tus juegos con diálogos dinámicos y conscientes del contexto. Aquí encontrarás guías y referencias para ayudarte a integrar NPC Builder en tus proyectos, usar nuestras API para conversaciones interactivas con NPCs y gestionar el contexto de tu juego (personajes, mundos, etc.) a través del sistema de NPC Builder.

<Columns cols={2}>
  <Card title="Guía de API de interacciones" icon="message-dots" href="es/interactions-api-guide">
    Aprende a usar la API de chat de NPC (endpoint `/interactions`) para mantener conversaciones dinámicas con tus NPCs. Cubre autenticación, formato de solicitud/respuesta y consejos para obtener los mejores resultados.
  </Card>
  <Card title="Guía de API de contexto" icon="database" href="es/context-api-reference">
    Gestiona el mundo de tu juego a través de la Context API. Crea y actualiza personajes NPC, mundos, regiones y ubicaciones de forma programática para mantener el estado del juego sincronizado con los diálogos de los NPCs.
  </Card>
</Columns>
