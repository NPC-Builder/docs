---
title: "Quickstart"
description: "Set up NPC Builder and run your first conversation"
---

<Info>
<strong>Prerequisites</strong><br/>
• NPC Builder account • Game created in dashboard
</Info>

<Steps>
  <Step title="Log in & create a game">
    Sign in at <a href="https://app.npcbuilder.com">app.npcbuilder.com</a> and create a new game project.<br/>
    Copy the shown <code>GAME_ID</code>.
  </Step>

  <Step title="Generate API token">
    In <em>Developers → API Tokens</em> click <strong>Generate Bearer Token</strong>.<br/>
    Store the JWT safely.  
    ```http
    Authorization: Bearer YOUR_TOKEN
    ```
  </Step>

  <Step title="Copy World ID">
    Expand the context tree and copy your <code>WORLD_ID</code>.  
    (Add more worlds any time.)
  </Step>

  <Step title="POST /interactions">
    ```bash
    curl -X POST https://app.npcbuilder.com/api/interactions \\
      -H "Authorization: Bearer YOUR_TOKEN" \\
      -H "Content-Type: application/json" \\
      -d '{
            "character_name": "Guardian",
            "game_id": "GAME_ID",
            "world_id": "WORLD_ID",
            "messages": [
              { "role": "user", "content": "Hello!" }
            ]
          }'
    ```
  </Step>

  <Step title="Read reply">
    ```json
    {
      "response": "Greetings, traveler!",
      "user_events": [],
      "character_events": []
    }
    ```
    Show <code>response</code> to the player; handle events as gameplay hooks.
  </Step>
</Steps>

<Note>Unity / Unreal plugins wrap these calls for you—see integration guides.</Note>
